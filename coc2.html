Watch Video Tutorials for COC-2: Set Up Computer Networks

What is COC-2?

This Core Competency covers basic network configuration setup and testing. Also covered are basic concepts and operations, including the difference between different types of networks and how IP Addressing is used. It also cover how to configure your router and accesspoint and how to properly create a network between your Server and Client

Course Features

2. How to create a Computer Network

3. Different types of Network and Topologies

4. Configuration of Router, Accesspoint, and Network

5. Sharing Folder and Workgroup
